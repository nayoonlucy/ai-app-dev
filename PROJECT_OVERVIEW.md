# 프로젝트 개요 및 개발 계기

## 📋 프로젝트 소개

**GitHub Portfolio Uploader**는 초급 개발자와 GitHub 사용이 어려운 사용자들을 위한 웹 기반 파일 업로드 도구입니다. 복잡한 Git 명령어 없이도 간단하게 GitHub에 파일을 업로드할 수 있도록 도와줍니다.

---

## 💡 개발 계기

### 문제 인식

현대 소프트웨어 개발에서 GitHub는 필수적인 도구가 되었습니다. 하지만 많은 초급 개발자와 학부생들이 GitHub를 활용하는 데 어려움을 겪고 있습니다:

#### 1. 기술적 진입 장벽
```
❌ 문제: Git 명령어의 복잡성
- git add, git commit, git push 등 다양한 명령어를 기억해야 함
- 터미널 사용에 대한 두려움
- 환경 설정의 복잡성 (SSH 키, 원격 저장소 연결 등)

✅ 해결: GUI 기반 웹 애플리케이션
- 마우스 클릭만으로 파일 업로드
- 명령어 없이 직관적인 인터페이스
- 자동화된 설정 및 관리
```

#### 2. 접근성 문제
```
❌ 문제: 컴퓨터 의존성
- Git은 주로 데스크톱 환경에서 사용
- 모바일에서는 접근이 어려움
- 컴퓨터가 없는 상황에서 파일 업로드 불가

✅ 해결: 웹 기반 반응형 애플리케이션
- 브라우저만 있으면 어디서나 접근 가능
- 모바일, 태블릿, 데스크톱 모두 지원
- 인터넷 연결만 있으면 즉시 사용 가능
```

#### 3. 효율성 문제
```
❌ 문제: 반복 작업의 번거로움
- 매번 동일한 명령어 입력
- 여러 파일을 개별적으로 업로드
- 파일 경로 관리의 복잡성

✅ 해결: 드래그 앤 드롭 및 배치 업로드
- 여러 파일을 한 번에 선택
- 드래그 앤 드롭으로 간편하게 업로드
- 자동 커밋 및 관리
```

## 🎯 프로젝트 목표

### 주요 목표
1. **진입 장벽 완화**: Git 명령어를 몰라도 GitHub 사용 가능
2. **접근성 향상**: 어디서나 접근 가능한 웹 기반 도구
3. **사용 편의성**: 직관적이고 간단한 사용자 인터페이스
4. **빠른 개발**: MVP 방식으로 핵심 기능 우선 구현

### 타겟 사용자
- 🎓 **학부생**: 포트폴리오 프로젝트를 GitHub에 업로드하고 싶은 학생
- 👨‍💻 **초급 개발자**: GitHub 사용의 진입 장벽에 부딪힌 개발자
- 🔧 **편의 추구자**: GitHub를 더 편리하게 사용하고 싶은 개발자

## 🚀 핵심 기능 (MVP)

### 1. 인증 시스템
- GitHub Personal Access Token 기반 인증
- 로그인 상태 유지
- 토큰 유효성 검증

### 2. 레포지토리 관리
- **조회**: 사용자의 모든 레포지토리 자동 로드
- **생성**: 새 레포지토리 즉시 생성
- **선택**: 업로드할 레포지토리 선택

### 3. 파일 업로드
- **드래그 앤 드롭**: 직관적인 파일 업로드
- **다중 선택**: 여러 파일 동시 업로드
- **자동 커밋**: GitHub에 자동으로 커밋 및 푸시

### 4. 사용자 경험
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **실시간 피드백**: 업로드 상태 및 결과 즉시 표시
- **직관적 UI**: 명확한 버튼과 안내 메시지

## 🛠 기술 스택

### 프론트엔드
- **React 18**: 사용자 인터페이스 구축
- **TypeScript**: 타입 안정성 및 코드 품질
- **Vite**: 빠른 개발 서버 및 빌드
- **Tailwind CSS**: 빠른 스타일링 및 반응형 디자인

### API
- **GitHub REST API**: 레포지토리 및 파일 관리
- **@octokit/rest**: GitHub API 클라이언트

### 선택 이유
- **빠른 개발**: React + Vite로 빠른 개발 속도
- **타입 안정성**: TypeScript로 런타임 오류 방지
- **반응형**: Tailwind CSS로 모바일 지원 용이
- **안정성**: GitHub REST API는 검증된 안정적인 API
